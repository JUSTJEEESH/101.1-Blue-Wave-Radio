=======================================
AFTER PUSHING - CLONE IN XCODE
=======================================

1. Open Xcode
2. Close any open projects
3. File → Clone... (or from Welcome screen)
4. Paste this URL:

   https://github.com/JUSTJEEESH/101.1-Blue-Wave-Radio.git

5. Click "Clone"
6. Choose where to save (e.g., Desktop)
7. Wait for it to download
8. Done! ✅


=======================================
NEED A PERSONAL ACCESS TOKEN?
=======================================

If you don't have a GitHub Personal Access Token:

1. Go to: https://github.com/settings/tokens
2. Click "Generate new token" → "Classic"
3. Name it: "Xcode Access"
4. Check: ✅ repo
5. Click "Generate token"
6. COPY THE TOKEN and save it (you won't see it again!)
7. Use this token as your password


=======================================
TROUBLESHOOTING
=======================================

❌ "Authentication failed"
   → Use Personal Access Token, not password

❌ "Remote already exists"
   → Run: git remote remove origin
   → Then try again

❌ "Permission denied"
   → Make sure you're logged into GitHub
   → Check your token has 'repo' permissions

❌ Still stuck?
   → Email the token/credentials issue screenshot
   → Or try running the script again
